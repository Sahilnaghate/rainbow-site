## Overview

Rainbow Masala Digital Platform is a premium B2B FMCG website built for **Rainbow Masala**, designed to serve:

- Hotels
- Restaurants
- Caterers
- Cloud Kitchens
- Institutional Buyers
- Distributors
- Retail Customers

The platform focuses on trust, product discovery, professional kitchen solutions, and lead generation while providing a modern digital experience.

---

## Vision

To build India's most trusted digital platform for professional kitchens.

The website is designed to showcase Rainbow Masala as a **Professional Kitchen Masala Partner**, emphasizing:

- Consistent Quality
- Reliable Supply
- Manufacturing Excellence
- Professional Solutions
- Business Growth

# Getting Started

## Clone

```bash
git clone https://github.com/Sahilnaghate/rainbow-site.git

cd rainbow-site
```

---

## Install

```bash
npm install
```

---

## Development

```bash
npm run dev
```

Application runs on

http://localhost:3003

---

## Production Build

```bash
npm run build
```

## Preview


# Project Philosophy

We don't build webpages.

We build reusable systems.

Every line of code should improve:

- Maintainability
- Performance
- Accessibility
- Scalability
- Developer Experience

---

# Golden Rule

Every feature should answer one question:

> **Does this make Rainbow Masala the most trusted digital partner for professional kitchens?**

If the answer is **No**, we don't build it.
