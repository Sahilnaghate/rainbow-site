# Purpose

The Design System ensures that every page, component, and interaction across the Rainbow Masala Digital Platform feels consistent, premium, and trustworthy.

It serves as the visual language of the brand.

This document is the single source of truth for UI and UX decisions.

---

# Design Philosophy

The website should never look like a typical FMCG catalogue.

Instead it should communicate:

- Premium Quality
- Trust
- Professionalism
- Simplicity
- Warmth
- Modern Manufacturing
- Business Confidence

The user should feel:

> "This company is professional enough to supply my business."

# Design Principles

## 1. Clarity

Users should understand every section within 3 seconds.

No visual clutter.

---

## 2. Simplicity

Remove unnecessary decoration.

Whitespace is a design element.

---

## 3. Consistency

Buttons

Spacing

Typography

Colors

Cards

Animations

must behave consistently.

---

## 4. Trust

Everything should reinforce credibility.

Large images

Certifications

Factory

Founder

Professional kitchens

---

## 5. Conversion

Every page should encourage one primary action.

Never overwhelm users with choices.

✓ Typography follows scale

# Golden Rule

Every screen should answer one question:

**"Would a hotel owner trust Rainbow Masala after seeing this page?"**

If the answer is **No**, redesign it.
