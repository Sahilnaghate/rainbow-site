# Vision

To build India's most modern B2B FMCG website that helps hotels, restaurants, caterers, distributors, institutional buyers, and retailers discover, evaluate, and purchase Rainbow Masala products with confidence.

The website should establish Rainbow Masala as a **Professional Kitchen Masala Partner**, focusing on reliability, consistency, and business value rather than traditional consumer-centric messaging.

We DO NOT build pages.

We build reusable systems.

Every component should be

- Reusable
- Accessible
- Responsive
- Performant
- Maintainable

---

# Core Principles

## Simplicity

Every page should communicate one clear objective.

---

## Performance First

Fast websites generate more enquiries.

Performance is considered a feature.

---

## Mobile First

Design for mobile.

Enhance for desktop.

---

## Accessibility

Every visitor should be able to use the website.

Follow WCAG guidelines wherever practical.

---

## SEO by Default

Every page should be indexable.

Every product should be searchable.

Every recipe should attract organic traffic.

---

## Reusability

Never duplicate components.

Never duplicate business logic.

Always extract reusable patterns.
